## ⚙️ Setup & Run

### 1. Clone the repository
```bash
git clone https://github.com/shirodkaraditya2003/DinePulse-Bangalore-Sentiment-Dashboard.git
cd "DinePulse-Bangalore-Sentiment-Dashboard/DinePulse Bangalore Sentiment Dashboard"
```

### 2. Install dependencies
```bash
pip install -r DinePulse/requirements.txt
```

### 3. Add your data files
Place the following CSV files inside the `data/` folder:
```
data/
├── zomato.csv               # Raw Zomato Bangalore dataset (from Kaggle)
├── sentiment_reviews.csv    # Pre-processed reviews with VADER scores
├── cleaned_restaurants.csv  # Cleaned restaurant metadata
└── restaurant_scorecard.csv # Health scorecard (generated by health_score.py)
```

> 📥 **Dataset**: [Zomato Bangalore Restaurants — Kaggle](https://www.kaggle.com/datasets/himanshupoddar/zomato-bangalore-restaurants)

### 4. Generate sentiment data (if running fresh)
```bash
python DinePulse/src/preprocess.py
python DinePulse/src/sentiment_model.py
python DinePulse/src/health_score.py
```

### 5. Launch the dashboard
```bash
streamlit run DinePulse/app/streamlit_app.py
```

Open your browser at **http://localhost:8501**
